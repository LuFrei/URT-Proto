## Unified Ray Tracing demo

This is simply an implementation of the sample code from the [Unified Raytracing API Unity Manual page](https://docs.unity3d.com/Packages/com.unity.render-pipelines.core@17.4/manual/UnifiedRayTracing/trace-camera-rays-full-sample.html).

To see it in action:
1. Open in Unity.
2. Enter play-mode.
3. Click on the cube gameobject on the scene to view it on the inspector.
4. Scroll down to the "Shoot Camera Rays" component.
5. Double-click the "RTTexture" output Texture.

Result should be displayed on the bottom viewport of the inspector window.
